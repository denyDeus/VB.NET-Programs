# VB.NET Programs

This repository contains a collection of **VB.NET programs** developed for learning, practice, and experimentation with the .NET framework.

The goal of this repository is to strengthen programming skills in **Visual Basic .NET** by building small applications, exploring programming concepts, and documenting practical solutions.

It serves as both a **learning archive** and a **portfolio of VB.NET development work**.

---

## 🎯 Objectives

- Learn and practice VB.NET programming
- Understand .NET application development
- Implement programming concepts through practical examples
- Build small desktop applications
- Improve coding structure and problem-solving skills
